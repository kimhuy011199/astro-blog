## Using next/font to apply a Google font globally

For this example, we will apply the [Google Open Sans](https://fonts.google.com/specimen/Open+Sans) font, which is a variable font. To start using the `next/font` package, we need to complete 3 steps:

1. Import the font
2. Configure the font object
3. Attach the font to the JSX code

We can do all 3 of these things in our `layout.js` file.

<BlockCode fileName="src/app/layout.js">

```jsx
import './globals.css';

//üëá Import Open Sans font
import { Open_Sans } from 'next/font/google';

//üëá Configure our font object
const openSans = Open_Sans({
  subsets: ['latin'],
  display: 'swap',
});

export default function RootLayout({ children }) {
  return (
    // üëá Attach font to global JSX node
    <html lang="en" className={openSans.className}>
      <body>{children}</body>
    </html>
  );
}
```

</BlockCode>

In the above code block, you can see we have imported our Google Open Sans font and configured our object with two properties.

## Install Tailwind CSS in our project

Now that we've updated our code to be usable with Tailwind CSS, we need to follow a few steps to install and configure it.

1. Install Tailwind CSS by opening a new tab and running:

<BlockCode>

```
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

</BlockCode>

2. Configure your template paths with the CSS variables:

<BlockCode fileName='tailwind.config.js'>

```jsx
// tailwind.config.js

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    './app/**/*.{js,ts,jsx,tsx,mdx}',
    './pages/**/*.{js,ts,jsx,tsx,mdx}',
    './components/**/*.{js,ts,jsx,tsx,mdx}',

    // Or if using `src` directory:
    './src/**/*.{js,ts,jsx,tsx,mdx}',
  ],
  theme: {
    extend: {
      fontFamily: {
        // üëá Add CSS variables
        sans: ['var(--font-opensans)'],
        mono: ['var(--font-roboto-mono)'],
      },
    },
  },
  plugins: [],
};
```

</BlockCode>

And that's it! Now we can use the `font-sans` and font-mono utility classes to apply the font to our elements. üëç
